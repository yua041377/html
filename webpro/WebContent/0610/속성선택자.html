속성을 이용한 접근선택자
	p{
		font-size : 1.5em;
	}
	title속성을 가지고 있는 p태그를 선택
	p[title]{
		background : yellow;
	}
	#p1{
		color : red;
	}

   alt 속성값이 커피인 img 태그 선택
   	img[alt="커피"]{
		border : 3px solid green;
	}
	#im1{
		border : 5px dotted hotpink;
	}
	id 속성값이 im2인 img선택
	img[id="im2"]{
		border : 10px inset green;
	}
	.cim{
		width : 150px;
		height : 150px;
		//id 속성이 class 속성보다 우선이므로 아래 스타일이 실행 되지 않는다.
		border : 10px dotted pink;
	}
