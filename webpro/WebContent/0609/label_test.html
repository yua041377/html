label : 문자, 인라인 요소, 크기조절이 안됨
	display : inline, block, inline-block - 인라인이면서 블럭 요소의 속성을 동시에 갖는다, 줄을 바뀌지 않고 크기 조절이 가능하다.
	
입력요소(input,textarea)의 값을 가져오거나 새로운 값으로 설정 - value
입력요소가 아닌 다른 요소에 값을 가져오거나 새로운 값으로 설정 - innerHTML, textContent

textarea : 여러줄 입력. 출력시에도 줄이 바뀌도록 한다.
varea = document.ff.area.value;
		// varea의 값을 엔터처리 한다.(입력시 엔터 : \r\n -> <br>)
		// replace(/old/g, new)
		varea = varea.replace(/\r/g,"").replace(/\n/g, "<br>");
